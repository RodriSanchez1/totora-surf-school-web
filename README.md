# Totora Surf School & Hostel - Landing Page

Landing page for Totora Surf School & Hostel located in Huanchaco, Peru. Built with React, TypeScript, and Tailwind CSS.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
